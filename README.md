## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`

## Project Title
Adaptive Relationship Engagement Platform (Bonding Moments)

---

## Problem Statement
Couples often struggle to maintain emotional engagement in daily life.
This platform helps couples stay connected through adaptive, AI-generated
daily activities that respond to emotional feedback and engagement patterns.

---

## Objective
To build an AI-powered relationship engagement platform that:
- Understands both partners through onboarding questions
- Generates personalized daily couple activities
- Collects feedback from both partners
- Adapts future tasks based on emotional intent and engagement

---

## Tech Stack
- Frontend: React + TypeScript (Vite)
- AI: Gemini API
- Hosting: Netlify
- Version Control: GitHub

---

## User Stories Completed
- User Registration & Authentication  
- Partner Connection via Invite Code  
- Onboarding Questions  
- AI-Based Daily Task Generation  
- Feedback Collection  
- Adaptive Task Personalization  
- Emotional Intent Understanding using AI  

---

## AI Usage
The Gemini API is used to:
- Generate personalized and emotionally safe daily couple activities
- Adapt tasks based on user feedback and engagement
- Infer emotional intent from responses to avoid repetitive or irrelevant tasks

AI responses are structured and validated to ensure respectful and relevant output.
